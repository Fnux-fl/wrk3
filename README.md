# wrk3

  **the corrected release of wrk2**

wrk2 is a modified version of wrk designed to generate a constant throughput load. Not only does it inherit the original design flaw, but it also introduces its own set of bugs.

wrk3 isn't perfect (and remains a bottleneck for testing the performance of fast web servers) because it's still too slow and resource-intensive, but at least it displays accurate values.

Please see this note: https://discourse.ubuntu.com/t/wrk-and-wrk2-bugs/81027?u=fnux.fl
